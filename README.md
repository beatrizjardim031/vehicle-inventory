# Vehicle Inventory App

### *This application is intended to manage the inventory for a used car dealership.  Users will use a menu to lookup vehicles or add vehicles to the list.*

## Features
### 1. Array capable of holding up to 20 vehicles;
### 2. List of all vehicles and its properties (color, makeModel, price and mileage);
### 3. Menu where you can search by vehicle properties and quit option; 

### I used Java and IntelliJ as IDE.
